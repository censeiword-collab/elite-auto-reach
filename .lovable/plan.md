

## Фавикон SUNMAX в сине-красном цвете

### Цель
Создать новый фавикон, который отражает брендовую идентичность SUNMAX с использованием синего (для "SUN") и красного (для "MAX") цветов, как в основном логотипе.

### Дизайн-решение
Создать иконку с буквой "S" (или "SM" для SUNMAX):
- **Синий цвет** (text-blue-500, ~#3B82F6): основной цвет для верхней части
- **Красный цвет** (text-red-500, ~#EF4444): акцентный цвет для нижней части или градиент
- **Темный фон** (графит/черный): соответствует премиальному стилю студии
- **Солнечный мотив**: может быть интегрирована идея солнца (лучи или круг), отражающая "SUN" в названии

### Техническая реализация
1. **Генерация изображения**: Использовать AI для создания фавикона размером 512x512px (PNG) с хорошей читаемостью в масштабе 16x16px, 32x32px и 180x180px
2. **Замена файла**: Заменить текущий `public/favicon.png` на новый файл с сине-красным дизайном
3. **Проверка**: HTML уже обновлена на использование `/favicon.png`, поэтому дополнительные изменения в `index.html` не требуются

### Файлы для обновления
- **Переписать**: `public/favicon.png` — новый фавикон в сине-красном цвете
