

# Виджет отзывов Яндекс Карт + исправление ошибок сборки

## 1. ReviewsSection.tsx -- замена на виджет Яндекс Карт

Заменим захардкоженные карточки отзывов на iframe-виджет Яндекс Карт с вашим ID организации **97524296927**.

**Что останется:**
- Заголовок секции ("Нам доверяют владельцы премиальных авто")
- Блок статистики (500+, 98%, 7 лет, 4.9)

**Что изменится:**
- Вместо 4 карточек отзывов -- iframe виджет: `https://yandex.ru/maps-reviews-widget/97524296927`
- Адаптивная высота: 380px на мобильных, 520px на планшетах, 680px на десктопе
- Кнопка "Все отзывы на Яндекс Картах" со ссылкой на карточку организации
- Удалим неиспользуемые импорты (`Star`, `Quote`)
- Удалим массив `reviews`

## 2. AdminPortfolio.tsx -- исправление ошибок TypeScript

6 ошибок сборки: `x.images` не существует на типе `PortfolioBlock` (т.к. `images` есть только у типа `gallery`).

**Исправление:** заменить `x.images` на `(x as any).images` в 6 местах (строки 400, 420, 438, 456, 475, 494).
