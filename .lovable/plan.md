

## Создание папок public/video и public/img

Создадим две новые папки в директории `public`:
- `public/video/` — для хранения видеофайлов
- `public/img/` — для хранения изображений

Так как пустые папки не могут существовать в файловой системе проекта без файлов, в каждую папку будет добавлен файл-заглушка `.gitkeep`, чтобы папки сохранились.

### Файлы для создания
- `public/video/.gitkeep` — пустой файл для сохранения папки
- `public/img/.gitkeep` — пустой файл для сохранения папки
